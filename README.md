## Description:

<p>PokeMon is a mini Pokemon encyclopedia and in it you will be able to see all existing Pokemon and relevant information about them, in an elegant and practice!</p>

<br>

# Getting Started:

## Install dependencies:

```bash
$ yarn
 or
$ npm i
```

## Build application:

```bash
$ yarn build
 or
$ npm run build
```

## Start application:

```bash
$ yarn dev
or
$ npm run dev
```
